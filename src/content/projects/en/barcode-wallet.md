# Barcode Wallet App

**Barcode Wallet** is a native iOS application designed to help users easily manage their gym membership passes and other barcode-based cards.

## Features

- **Scan & Save**: Quickly scan barcodes using the camera and save them to the app.
- **Generate Barcodes**: Automatically generate barcodes from saved data.
- **Card-Based Layout**: A modern, clean UI for easy navigation.
- **Ticket View**: A detailed view for each pass, mimicking a real ticket.

## Tech Stack

- **Swift**: The primary programming language.
- **SwiftUI**: For building the user interface.
- **AVFoundation**: For barcode scanning capabilities.
